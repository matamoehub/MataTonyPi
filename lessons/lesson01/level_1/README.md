# Lesson 1 - Hello TonyPi

## Goal

Meet TonyPi through the central V2 robot library.

## What students should do

- make TonyPi talk
- move TonyPi's head
- wave and greet
- combine speech and animation into a short performance

## Library Rule

Students only use:

```python
from student_robot_v2 import bot
```

They should not import vendor TonyPi modules directly.
