# Lesson 15 - Wireless Joystick Control

## Goal

Learn the TonyPi joystick button layout, dance shortcuts, and controller modes so students can understand what the wireless controller does.

## What students will use

- `myRobot.controller.buttons()`
- `myRobot.controller.dance_buttons()`
- `myRobot.controller.modes()`
- `myRobot.controller.summary()`
- TonyPi actions that match the joystick buttons

## Teaching Note

This lesson is an instruction and testing lesson. The notebook explains the controller mapping and links the physical joystick buttons to action groups and student API commands.
